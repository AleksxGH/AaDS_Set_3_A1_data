# Алгоритмы и структуры данных. Set 3. Задача A1. Экспериментальные данные
Данный репозиторий содержит csv файлы с экспериментальными запусками **алгоритма Монте-Карло для подсчета площади пересечения трех окружностей** с заданными параметрами:
- с1: с центром в точке (1, 1) и радиусом 1,
- c2: с центром в точке (1.5, 2) и радиусом √5/2
- c3: с центром в точке (2, 1.5) и радиусом √5/2
- 
В файле **narrow_data.csv** находятся данные, полученные при рассмотрении узкой области генерации точек (прямоугольник содержит только саму область пересечения).
В файле **wide_data.csv** находятся данные, полученные при рассмотрении широкой области генерации точек (прямоугольник содержит все три окружности).

Формат данных в файлах следующий:
- **N** — количество случайно сгенерированных точек в области (целое число, 100 <= N <= 100000),  
- **ApproxArea** — приближённая оценка площади пересечения трёх кругов (вещественное число),  
- **RelError** — относительная ошибка вычисления площади по сравнению с её точным значением (вещественное число, без знака процента).
